# chatapp

## Running 

Run with Strealit 

- app/chat/app.py <== main chat app
- app/admin/app.py <== admin app


- chat/app.py Query String:
    - mode = chat | search
    - appid = the id of the registered application
    - userid = unique id for the user (from qualtrics)
